# Dossier Images

Ce dossier contient les images des produits LARQ.

## Structure recommandée

Placez vos images de produits dans ce dossier avec les noms suivants :

- `bottle-swig-top.jpg` - LARQ Bottle Swig Top
- `bottle-filtered-swig-top.jpg` - LARQ Bottle Filtered Swig Top
- `bottle-purevis.jpg` - LARQ Bottle PureVis™
- `bottle-filtered-flip-top.jpg` - LARQ Bottle Filtered Flip Top

## Formats acceptés

- JPG / JPEG
- PNG
- WebP

## Tailles recommandées

- Largeur minimale : 800px
- Ratio recommandé : 4:3 ou 16:9
- Poids optimal : < 500KB par image

## Notes

Les images seront automatiquement chargées par le composant ProductCard. Si une image n'est pas trouvée, le système utilisera automatiquement le modèle 3D comme fallback.
